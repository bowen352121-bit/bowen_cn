景教留言板 · Firebase 配置

按步骤完成配置后，访客即可用 Google / GitHub 登录留言。

步骤 1 · 创建 Firebase 项目

  1. 打开 Firebase 控制台，新建项目（建议名称：bowen-guestbook）
  2. 项目 → 项目设置 → 你的应用 → 添加 Web 应用（ 图标）
  3. 复制下方 6 项配置，填入本页表单
打开 Firebase 控制台

步骤 2 · 启用登录方式

  1. Authentication → Sign-in method → 启用 Google
  2. 启用 GitHub（需先在 GitHub 创建 OAuth App）：
    • GitHub → Settings → Developer settings → OAuth Apps → New
    • Homepage URL：
    • Authorization callback URL：https://你的项目ID.firebaseapp.com/__/auth/handler
    • 将 Client ID / Secret 填入 Firebase GitHub 登录配置
  3. Authentication → Settings → Authorized domains → 添加 bowen352121-bit.github.io
打开 GitHub OAuth 设置

步骤 3 · 创建数据库与存储

  1. Firestore Database → 创建数据库（选择就近区域）
  2. Storage → 开始使用（默认 bucket 即可）
  3. 部署安全规则（有 Node.js 时运行 npx firebase deploy --only firestore:rules,storage；或在控制台 Rules 标签页手动粘贴 firebase/firestore.rules 与 firebase/storage.rules 内容）
